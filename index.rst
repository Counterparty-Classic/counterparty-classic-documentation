Welcome to the Official Counterparty Documentation
==================================================

`Counterparty`_ is a **peer-to-peer financial platform** and a distributed, open-source Internet protocol built on top of the **Bitcoin blockchain** and network.

The code is open source, and `available on github`_.

.. _Counterparty: http://counterparty.io/
.. _available on github: https://github.com/CounterpartyXCP

* :ref:`counterparty-docs`
* :ref:`technology-docs`
* :ref:`counterwallet-docs`

Information about development is also available:
